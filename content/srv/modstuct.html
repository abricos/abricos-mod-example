Структура модуля "Example"


content/                - файлы контента (стартовые кирпичи), с них начинается сборка ответа сервера
       *.html
brick/                  - строительный материал, могут быть использованы в шаблоне многократно
     *.html
includes/               - скрипты модуля
        manager.php     - менеджер модуля
        shema.php       - схема таблиц модуля
        helloworld.php  - скрипт используемый в кирпиче brick/helloworld.html
js/                     - JS компоненты для пользовательского интерфейса на стороне клиента 
  langs/                - языковые фразы
       *.js
  *.js                  - JavaScript файлы
  *.htm                 - шаблоны интерфейса
  *.css                 - CSS интерфейса
module.php              - главный скрипт модуля
[mod]example:up[/mod]
